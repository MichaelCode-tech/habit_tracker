# 🌿 Habit Tracker

A beautiful, minimalist habit-tracking application built with Flutter and Isar Database. Designed to help you build consistency through visual progress and streaks.

## ✨ Features

- **📊 Visual Progress**: Track your long-term consistency with an interactive GitHub-style HeatMap.
- **🔥 Daily Streaks**: Stay motivated with daily streak counters for every habit.
- **🌓 Adaptive Theming**: Seamlessly switch between elegant Light and Deep Midnight Dark modes.
- **📉 Analytics**: View your total habit completions on a monthly and yearly basis.
- **💾 Offline First**: Powered by Isar, a high-performance NoSQL database that works entirely offline.
- **🛠 Modern UI**: Clean, minimalist design with smooth animations and intuitive gestures (swipe to edit/delete).

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest version)
- Dart SDK

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/habit_tracker.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd habit_tracker
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the build runner (for Isar generation):**
   ```bash
   dart run build_runner build
   ```

5. **Run the app:**
   ```bash
   flutter run
   ```

## 🛠 Tech Stack

- **Framework:** [Flutter](https://flutter.dev)
